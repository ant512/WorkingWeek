WorkingWeek V1.00
-----------------

  WorkingWeek is a very small library used for calculating dates based on a
  user-defined week of working shifts.
  
  See http://www.bitbucket.org/ant512/workingweek for more information and a 
  short tutorial.
  
Requirements
------------

  Written with .NET4.  It may be possible to build from source for earlier
  versions of the framework.
